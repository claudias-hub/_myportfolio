· End to End Design ·

Eine einfache Idee zu einer funktionierenden Live-Website entwickeln.

Diese Website ist eine kleine Sammlung meiner Arbeit als Frontend-Webentwicklerin. Du kannst durch einige Projekte stöbern, den Code sehen und die Live-Versionen ausprobieren.

Die Projekte reichen von Single-Page-Apps bis zu Full-Stack-Beispielen mit einer eigenen API. Die meisten nutzen HTML, CSS und JavaScript im Frontend, manchmal mit React. Einige beinhalten auch ein Node.js- und Express-Backend, Routing, Authentifizierung oder Daten von externen APIs.

Wenn du sehen möchtest, wie ich arbeite, schau dir meine Projektgalerie an oder schick mir eine kurze Nachricht über das, was du brauchst.
