...und ich liebe Fotografie als Hobby!

Ich arbeite hauptsächlich mit HTML, CSS und JavaScript. Einige meiner Projekte nutzen React für interaktivere Frontends, und andere beinhalten einfache Backends mit Node.js und Express. Wie ich gerne arbeite

  • Mit Klarheit beginnen: Ziel, Nutzer und Hauptabläufe definieren, bevor Features hinzugefügt werden.
  • Basics zuerst: sauberes Layout, lesbarer Text, guter Kontrast und funktionierende Navigation.
  • In kleinen Schritten bauen: etwas liefern, das funktioniert, dann basierend auf Feedback verfeinern.
  • Verständlich halten: klare Namen, einfache Strukturen und kleine, fokussierte Komponenten bevorzugen.
  • Wartbar machen: Git nutzen, kleine Commits und unkomplizierte Deployment-Setups.

Du kannst einige Beispiele dieses Ansatzes in meinen Projekten erkunden.
