Soy Claudia, desarrolladora web front‑end, vivo en Berlín y soy originalmente de Venezuela.

...¡y me encanta la fotografía como hobby!

Trabajo principalmente con HTML, CSS y JavaScript. Algunos de mis proyectos usan React para crear frontends más interactivos, y otros incluyen backends sencillos con Node.js y Express. Mi hobby de la fotografía agudiza mi ojo para la composición y el contraste, y eso se refleja en mis decisiones de UI.

Habilidades y herramientas

Estas son las principales tecnologías que he usado en los proyectos de este portafolio:

Frontend esencial

HTML5, CSS moderno (Flexbox, Grid), diseños responsive, manejo de formularios.

JavaScript y React

JavaScript (ES6+), React para UIs basadas en componentes, React Router básico y manejo de estado.

APIs y datos

Consumo de APIs REST, creación de endpoints REST sencillos con Node.js y Express, trabajo con datos JSON.

Ejemplos full‑stack

Proyectos que combinan una API personalizada (Node.js/Express, MongoDB) con un cliente frontend separado (React o JavaScript vanilla).

UI estilo móvil

Experiencia con una app de chat creada con React Native/Expo, enfocada en interfaces simples y amigables para móvil.

Flujo de trabajo y entrega

Git, GitHub, branching, despliegues con GitHub Pages/Vercel, documentación básica en READMEs.

Cómo me gusta trabajar

  • Empezar con claridad: definir el objetivo, los usuarios y los flujos principales antes de añadir funcionalidades.
  • Priorizar lo básico: layout limpio, texto legible, buen contraste y navegación funcional.
  • Construir en pasos pequeños: entregar algo que funcione y luego mejorar con feedback.
  • Mantenerlo entendible: preferir nombres claros, estructuras simples y componentes pequeños y enfocados.
  • Que sea mantenible: usar Git, commits pequeños y despliegues directos.

Puedes ver ejemplos de este enfoque en mis proyectos.
